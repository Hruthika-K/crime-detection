# **App Name**: Terra-Civitas

## Core Features:

- Login: Allow users to log in using Firebase Authentication with email and password.
- Dashboard: Display a dashboard with a list of crime alerts, including title, location, and timestamp.
- Alert Details: Display detailed information about each crime alert when selected.
- Real-time Updates: Use dummy data to show as a placeholder.

## Style Guidelines:

- Primary color: Deep red (#B71C1C) to convey urgency and importance, in alignment with the app's purpose.
- Background color: Light gray (#EEEEEE) for a clean, modern look.
- Accent color: Dark red (#7f0000) for highlighting critical information and interactive elements.
- Body and headline font: 'Inter', a sans-serif font that provides a modern, clean look.
- Use universally recognizable icons for alerts and navigation.
- Design a clean, intuitive layout with a focus on easy navigation and information hierarchy.
- Incorporate subtle animations for transitions and loading states to improve user experience.
